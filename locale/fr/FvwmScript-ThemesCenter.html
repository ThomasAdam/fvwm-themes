Documentation du Themes Center

Le Thèmes Center est un FvwmScript qui vous permet de sélectionner vos thèmes, de modifier votre configuration générale et de gérer vos session FVWM Themes. Vous pouvez le lancer via le menu Themes Management / Themes Center GUI.

1. Sélecteur de Thèmes

La liste de tous les thèmes se trouve en haut à gauche. Si vous sélectionner l'un d'entre eux la liste des composantes de ce thème est affichée en haut à droite. Alors, vous pouvez ou bien sélectionner toutes les composantes d'un thème en utilisant le bouton Sélectionner de gauche, ou bien sélectionner seulement les composantes de type Look en utilisant le bouton Le Look, ou bien vous pouvez sélectionner une seule composantes en sélectionnant la composant et en utilisant le bouton Sélectionner de droite. Vous pouvez avoir des informations sur un thème en utilisant le bouton LisezMoi (les autres boutons de cette ligne ne fonctionnent pas encore).

Votre configuration courante et tous les changements effectués sont reflétés dans la liste que l'on voit en bas. La première colonne donne la liste de toutes les composantes, la deuxième colonne vous indique quelle thème est utilisé pour la composante et la troisième colonne (vide lors du démarrage du Script) vous indique les changements que vous avez effectués et qui n'ont pas été encore appliqués. Finalement, la dernière colonne peut contenir Var, Opt ou V/O ce qui indique que la composante du thème (Courant ou celui que vous avez sélectionner) a des options ou des variantes.

Si une composante a des variantes ou des options vous pouvez les modifier en la sélectionnant (avec la liste du bas) et en utilisant le bouton Propriétés (à droite de la liste). De plus, vous pouvez annuler vos changements avec le bouton Rétablir, vous pouvez bloquer une composante en utilisant le bouton bloquer (ainsi si vous sélectionner toutes les composantes d'un thème les composantes bloquées ne sont pas affectées). Le bouton Enlever n'est pas encore implémenté.

Maintenant, pour appliquer vos changement utilisez le bouton Appliquer. Le bouton Rafréchir est similaire au bouton Appliquer dans le sens que la configuration est complètement rechargée, mais vos changements ne sont pas appliqués. Le bouton Mise à Jour reconstruit les listes du Theme Center; ceci est utile si vous avez appliqué des changements avec un autre moyen de configuration comme le menu Theme Management ou en utilisant fvwm-themes-config en ligne de commande.

2. Configuration Générale

Vous pouvez avec cet outil de configuration effectuer les mêmes changements qu'avec le Theme Management -> current -> settings -> sous menu (vous pouvez aussi utiliser la Form et ou le Gtk ThemeSettings accessible via le menu Fvwm Configuration). La liste du haut vous indique votre configuration (actuelle et future). Pour modifier votre configuration sélectionnez une composante, et ainsi la liste des choix possibles pour cette composante vous sera proposée. Les autres boutons (en particuliers ceux du bas) fonctionnent exactement comme pour le Sélecteur de Thème.

Note Importante: Si vous utilisez Appliquer les changements serons appliqués pour les deux outils de configurations, le Sélecteur de Thèmes et la Configuration Générale.

3. Sessions FVWM Themes

FVWM Themes a son propre concept de Session. Ceci est utile pour pouvoir utiliser différente configuration avec FVWM Themes. Pour en savoir plus consultez la page de manuel de fvwm-themes-start. Cet outil de configuration a déjà une mini aide intégrée et est très simple à utiliser.

Plutôt que des explications générales donnons un exemple d'utilisation. Si vous voulez tester une configuration sans modifier votre configuration courante, créez une session test (ou réutilisez la si vous l'avez déjà crée) et utilisez votre configuration courante (en général main). Après avoir crée test cliquez sur Appliquer. Alors, vous pouvez utiliser librement le Sélecteur de Thèmes et la Configuration Générale sans modifier votre configuration main. Vous pouvez ensuite toujours revenir à votre session main ou bien si la nouvelle configuration test vous plais vous pouvez éditer votre configuration main pour qu'elle devienne votre configuration par défaut.

4. Divers

Le Script a quelques options. Il est lancé comme suit par le menu Theme Management:

FvwmScript FvwmScript-ThemesCenter --text-colorset 36 --viewer-colorset 33

I.e., la colorset 36 est utilisée pour les "List" et les "TextField" et la colorset 33 est utilisée pour cette aide (la colorset 35 est utilisé pour le reste comme pour tous les Scripts de FVWM Themes). La seule autre option est --lang xx où xx sont les deux premières lettres d'une LANG UNIX, comme "ru_RU.koi8r". Par défaut la variable d'environnement LANG ou "en" est utilisée. Les langues actuellement supportées sont en, fr et ru.
