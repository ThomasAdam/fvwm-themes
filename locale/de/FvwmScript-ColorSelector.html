Farben-Auswahl Dokumentation

Um eine Farbe zu waehlen, sollten Sie die 4. Farb-Palette nutzen. Die ersten drei Paletten ändern die 4. Palette.

Wenn Sie eine Farbe aus der ersten Palette wählen, erzeugen Sie einen Gradient mit dieser Farbe in der Art Schwarz-Farbe-Weiss.

Wenn Sie eine Farbe aus der zweiten oder dritten Palette wählen, wird die 4. Palette zum Gradient von dieser Farbe zur gewählten Farbe.

Außerdem, das Klicken in die gewählte Farbe hat den selben Effekt wie das wählen einer Farbe aus der ersten Palette.

Am Ende können Sie die Werte von Rot, Grün und Blau der gewählten Farbe mit dem mini Rollbalken ändern.

Fehlerreport nach: fvwm-themes-devel@lists.sourceforge.net.
